Open Source Developers Club - Modiin

The OSDC Modiin is going to have regular monthly meeting starting February 2014 in the Modiin Hub. If you are interested, please subscribe to our mailing list.

Topics

Some of the topics that will be addressed during the meetings include

  • Web application development using Perl, Python, Ruby, PHP, Node.js etc.
  • Web front-end development using HTML5/CSS3/JavaScript.
  • Mobile application development using HTML5/CSS3/JavaScript and PhoneGap.
  • NoSQL database
  • Test Automation
  • Using Linux
  • etc...
Come, join us on the mailing list and suggest your topics.
